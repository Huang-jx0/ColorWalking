# ColorWalking 产品需求文档（PRD）

## 1. 产品定位
- 产品名：ColorWalking
- 核心理念：每天抽取一个幸运色，给用户一个轻量、积极、可持续的好心情仪式感。
- 目标平台：本地 iOS 与 Android App，同时有品牌官网。

## 2. 目标用户
- 需要快速情绪提振的普通大众。
- 喜欢轻互动、低学习成本、可每天重复使用的工具型用户。

## 3. 核心功能
1. 幸运色转盘
- 多颜色扇区组成圆盘。
- 点击圆盘任意区域可触发抽取。
- 点击圆盘中心按钮（文案：再抽一次）也可触发抽取。
- 抽取时圆盘围绕中心平滑旋转，圆心位置固定不晃动。

2. 抽取结果展示
- 抽中颜色高亮。
- 展示颜色名、HEX 值、当日鼓励文案。

3. 每日记录
- 保存抽取历史（日期、颜色、时间戳）。
- 当天可重复抽取，保留最近记录。

## 4. 非功能需求
- 动画稳定：60fps 目标，避免抖动/偏轴旋转。
- 结构可复用：颜色配置、抽取算法、文案策略独立为共享模块。
- 可扩展：后续可新增节日主题盘、用户自定义色盘。

## 5. 品牌与设计要求
- 视觉关键词：简洁、干净、高级、温暖、原创。
- 独特元素：五彩斑斓的小羊卷（作为品牌 IP 装饰元素）。
- 版权声明：页面与应用内明确原创与版权归属。

## 6. 验收标准
1. iOS/Android 可本地运行。
2. 转盘可通过点击外圈和中心按钮触发随机抽取。
3. 圆盘旋转时中心固定，不出现晃动。
4. 官网可运行并展示品牌、功能、下载指引与版权信息。
