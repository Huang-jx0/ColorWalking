# ColorWalking 页面结构

## App 信息架构
1. 首页（转盘）
- 顶部：今日幸运色标题
- 中部：转盘 + 固定指针 + 中心按钮“再抽一次”
- 下部：抽取结果卡片（颜色名/HEX/鼓励语）
- 底部：最近 5 次历史

2. 品牌说明页（可后续扩展）
- 品牌故事
- 五彩斑斓的小羊卷 IP 介绍
- 版权与原创声明

## 官网信息架构
1. Hero 首屏
- ColorWalking 品牌名
- Slogan：Walk in color, walk in mood.
- 主 CTA：立即开始抽取幸运色

2. 产品亮点
- 一键抽色
- 平滑旋转
- 每日仪式感

3. IP 区块
- 五彩斑斓的小羊卷视觉
- 创作理念

4. 版权区
- 原创声明
- 版权归属说明
